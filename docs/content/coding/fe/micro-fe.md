解决问题： 遗留代码的升级。

## 是什么
各个前端应用还可以独立运行、独立开发、独立部署。

微前端的使命我认为是：「让天下没有短命的控制台」。
事实上如果所有的 web 技术栈能做到统一，所有 library 的升级都能做到向下兼容，我们确实就不需要微前端了。  —— 鲁迅
